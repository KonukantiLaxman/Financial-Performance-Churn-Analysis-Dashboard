## Financial Performance & Customer Churn Analysis Dashboard

A KPI-driven Excel dashboard project focused on revenue optimization, churn reduction, and financial risk monitoring for accounting and subscription-based businesses.

# Dataset Used

The dataset includes multi-year financial and customer-level records (2022–2024) covering:
-Revenue & Profit data\
-Industry & Regional performance\
-Subscription plans\
-Customer churn status\
-Tenure segmentation\
-Payment status (Paid / Pending / Overdue)\
-Revenue-at-risk metrics

# Business Questions & KPIs

This dashboard answers the following key business questions:

# Financial Performance:

-What is the total revenue, total profit, and overall profit margin?\
-Which industry generates the highest and lowest profit margin?\
-Which region contributes most to profit?\
-Which subscription plan is most profitable?

# Customer Churn Analysis

-What is the overall churn rate?\
-Which tenure segment has the highest churn?\
-Which subscription plan has highest churn risk?\
-How does monthly churn trend over time?

# Risk Analysis

-What percentage of revenue is at risk (overdue)?\
-Which region has highest overdue exposure?\
-How many customers are loss-making?\
-How does churn impact profitability?

# Process

# Step 1 – Data Collection

-Collected structured financial and customer-level data (2022–2024).

# Step 2 – Data Cleaning

-Removed duplicates\
-Standardized industry & subscription categories\
-Validated revenue, profit & margin calculations\
-Handled missing values\
-Maintained data integrity using SQL joins

# Step 3 – Data Modeling

-Created calculated columns (Profit Margin, Year, Month)\
-Built churn segmentation logic\
-Designed revenue-at-risk monitoring model

# Step 4 – Dashboard Development

-Built 3 interactive Excel dashboards\
-Used Pivot Tables, Slicers & Dynamic KPIs\
-Designed visual layout for executive-level reporting
